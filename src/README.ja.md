# src/

ソースコード。スプリント2以降で[WBS](../docs/02-scope/wbs.ja.md)および[技術設計書 (TDD)](../docs/07-design/tdd.ja.md)に従って実装を開始する。

想定ディレクトリ構成（ADR-005のクライアントスタック選定によって変更の可能性あり）:

```
src/
├── client/           # プッシュトゥトーク (PTT) 入力 + 詳細パネル表示
├── server/
│   ├── api/          # HTTPエンドポイント
│   ├── orchestrator/ # インテント解析 + スキルディスパッチ
│   ├── skills/       # Skillインターフェースを実装する各スキルモジュール
│   ├── stt/          # 音声認識 (STT) プロバイダのアダプター
│   ├── tts/          # 音声合成 (TTS) プロバイダのアダプター
│   └── llm/          # LLMプロバイダのアダプター + フォールバック
└── shared/           # Response契約型、共通ユーティリティ
```
