## このダッシュボードの更新方法

週次、ステータスレポート作成と同時に:
1. `基準日`を更新
2. フェーズゲートを越えた場合は `[●]` の位置を移動
3. RAGと「今週」表を更新
4. 未解決事項の件数を更新
5. 順位が変わった場合はトップ3リスクを調整

その週のステータスレポートと同じコミットにまとめる。
